# Xophz Midnight Nerd

> **Category:** Wizard's Tower · **Version:** 0.0.1

Use this 24/7 support ticketing system to request support from the author directly.

## Description

**Midnight Nerd** is a planned support and ticketing application for the COMPASS platform. It is designed to provide direct communication channels between site administrators and the development/support team. 

## Requirements

- **Xophz COMPASS** parent plugin (active)
- WordPress 5.8+, PHP 7.4+

## Status

🔴 **In Development** - Currently scaffolded with base WordPress plugin Architecture. Full ticketing functionality has not yet been implemented.

## Installation

1. Ensure **Xophz COMPASS** is installed and active.
2. Upload `xophz-compass-midnight-nerd` to `/wp-content/plugins/`.
3. Activate through the Plugins menu.
4. Access via the My Compass dashboard → **Midnight Nerd**.

## Frontend Routes

| Route | View | Description |
|---|---|---|
| `/midnight-nerd` | Dashboard | Support ticketing interface (planned) |
